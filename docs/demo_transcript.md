# Safety demo transcript

Generated by `uv run python demos/demo_safety.py`.

The demo drives the **real compiled LangGraph** end to end. Two boundaries are
stubbed so it is deterministic and needs no network — the **LLM** (intent
classification for the ambiguous case, SQL generation) and **BigQuery**
(dry-run + execute). Everything being demonstrated is the genuine production
code path:

1. **PII / raw-table extraction attempt** → the intent gate refuses via a
   deterministic rule, with no LLM call, so the refusal cannot be talked around.
2. **Analysis answer where masking fires** → a clean query passes the layer-2
   PII column denylist and runs; the returned free-text note contains an email,
   phone, and street address, and the deterministic layer-3 output mask scrubs
   all three *before* the report is produced.
3. **Full delete-with-confirmation** → a natural-language delete resolves an
   owner-scoped candidate set, the graph pauses on `interrupt()`, a wrong answer
   cancels cleanly, the exact typed phrase executes the delete, and an immutable
   audit record is written. The other manager's report is never in scope.
